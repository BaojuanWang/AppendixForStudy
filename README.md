# Learning Less While Asking More: How LLM-Based Assistance Shapes Vulnerability Learning

Supplementary materials for the paper submitted to CCS 2026.

## Contents

- **Qualtrics Survey.pdf**: Survey instrument used in both the main study and the pilot study.
- **Detailed Rubrics.pdf**: Scoring rubrics for all assessment dimensions and vulnerability types.
- **Codebook.pdf**: Qualitative codebook for open-ended response coding.
- **Variable Table.pdf**: Variables used in the learning gain analysis (RQ1).
- **Code Snippets/**: Pre/post-test code snippet stimuli. See README inside for details.
- **PlainLLM-VulnScaffold/**: Materials for both the plain LLM and VulnScaffold conditions. See README inside for details.
